## ✅ 一、交易逻辑核心：早盘抢筹 + 主力确认

这个策略可以概括为以下三步：

### 1️⃣ 集合竞价阶段判断主力意图（9:20 前后）

* 看“**涨停价挂单金额（封单金额）**”是否巨大（比如超过1亿元）
* 一般出现在题材热点（如军工、AI）中
* 越早出现（比如9:15封单就排在涨停价），说明越可能是“VIP隔夜单”

### 2️⃣ 开盘前做预判（9:25）

* 看集合竞价最终成交价是否接近涨停价
* 看换手率是否高（预判资金活跃度）
* 初步决定是否将该票加入 **“关注/准备买入列表”**

### 3️⃣ 开盘后快速确认（9:30\~9:33）

* 观察是否持续有大额主动买入资金流入（比如每分钟5000万以上）
* 看是否封涨停或接近涨停，且封单继续放大
* 一旦确认主力控盘 → 立即下单（跟主力抢筹）

## ✅ 二、执行例子：以“奥普光电”为例（模拟）

### 📆 某日交易情景：

#### 🕒 9:20

* 奥普光电 涨停价 60.00 元
* 涨停封单金额达到 **1.5 亿元**（从9:15就挂上去）
* 属于“军工概念股”，前日未涨停（非连板）

👉 判断：**VIP隔夜板，主力有强烈封板意图**

---

#### 🕒 9:25

* 集合竞价成交价为 59.95（接近涨停）
* 成交量高，换手达到 2%
* 同板块“航天宏图”“北方导航”等也有涨停封单

👉 判断：**强板块联动 + 集合竞价确认主力参与**

---

#### 🕒 9:30\~9:32

* 开盘后立即拉升，触及涨停 60.00 元
* 盘中买一挂单持续放大，从 8000万 涨到 2.5亿
* 无明显砸盘，盘面情绪高涨，其他军工股也陆续封板

👉 判断：**主力控盘无疑，强势涨停板**

## ✅ 三、常见风险点

| 场景        | 风险说明                   |
| --------- | ---------------------- |
| 主力诱多封单后撤单 | 集合竞价封单是“假封单”，9:30后直接低开 |
| 开盘无持续大单流入 | 主力未继续控盘，涨停打不开或者回落      |
| 大盘情绪转弱    | 板块涨停开板，导致个股“高开低走”      |
