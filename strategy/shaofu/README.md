# 少妇战法策略 - 聚宽回测版本

基于原始"少妇战法"策略逻辑，专为聚宽平台回测设计的完整实现。

## 策略核心逻辑

### 两种买入信号

**B1信号 - 回踩反包**
- 前一日：首阳（阳线 + 放量2倍以上）
- 当日：回踩（低点触及布林中轨或MA10）+ 缩量（≤昨日成交量*0.6）+ 反包（收阳线）
- 执行时间：14:50

**B2信号 - 突破前高**
- 前一日：首阳（但不是B1信号）
- 当日：突破前一日高点
- 执行时间：09:35

### 风控机制

**止盈**
- 价格达到前高*1.02 或 布林上轨（取较高者）

**止损**
- 价格跌破B1低点*0.98 或 MA5（取较低者）

**时限**
- 持有超过5个交易日强制卖出

## 选股策略

### 行业热度筛选
1. 计算申万一级行业近5日中位涨幅
2. 选择涨幅前3的行业
3. 从热门行业中筛选个股

### 基础过滤条件
- 排除ST股票
- 排除停牌股票
- 排除科创板(688)、创业板(300)、北交所(8、4开头)
- 排除上市不足60天的新股

### 规模与流动性
- 流通市值 ≤ 150亿
- 近10日日均成交额 ≥ 1亿元
- 最终候选池：80只股票

## 参数配置

```python
g.N_BOLL = 20           # 布林带周期
g.K = 2                 # 布林带标准差倍数
g.VOL_RATIO = 2.0       # 首阳放量倍数
g.SHRINK = 0.6          # 回踩缩量倍数
g.TAKE1 = 1.02          # 止盈：前高上浮2%
g.SL = 0.98             # 止损：B1低点下浮2%
g.MAX_HOLD_DAYS = 5     # 最长持有天数
g.MAX_POS = 0.9         # 总仓位上限90%
g.POS_PER_STOCK = 0.2   # 单票目标仓位20%
g.MAX_STOCKS = 5        # 最大持股数量
```

## 费用设置

- **佣金**：万分之2（双向）
- **印花税**：千分之5（仅卖出）
- **滑点**：0.3%（价格相关滑点）
- **最小佣金**：5元

## 使用方法

### 1. 直接复制到聚宽
将 `shaofu.py` 的全部代码复制到聚宽策略编辑器

### 2. 回测设置建议
- **起始资金**：100万
- **回测周期**：建议包含牛市和熊市
- **基准**：中证500指数 (000905.XSHG)
- **频率**：日频

### 3. 参数调优
可根据不同市场环境调整关键参数：
- `VOL_RATIO`: 放量倍数（1.5-3.0）
- `SHRINK`: 缩量倍数（0.5-0.8）
- `TAKE1`: 止盈倍数（1.01-1.05）
- `SL`: 止损倍数（0.95-0.99）

## 策略特点

### 优势
- **精确的技术形态识别**：首阳、回踩、反包逻辑清晰
- **双重买入机制**：B1和B2信号互补
- **严格风控**：止盈、止损、时限三重保护
- **行业轮动**：动态跟踪热门行业
- **适度分散**：最多5只股票，降低个股风险

### 适用环境
- **牛市/震荡市**：效果较好
- **主题轮动期**：表现优异
- **熊市**：需谨慎使用或暂停

### 注意事项
- 策略依赖技术分析，需关注市场整体趋势
- 行业轮动频繁时换手率可能较高
- 建议结合市场环境判断启停时机
