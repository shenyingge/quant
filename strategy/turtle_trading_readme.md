# 海龟交易法止损系统说明

## 功能概述

已在 `quick_roll.py` 策略中集成海龟交易法止损系统，专注于风险控制，不包含加仓逻辑。

### 1. ATR（真实波动幅度）计算
- 使用20日ATR作为波动性度量
- ATR = 最近20天真实波幅的平均值
- 真实波幅 = max(当日高-低, |当日高-昨收|, |当日低-昨收|)

### 2. 双重止损机制

#### 海龟动态止损
- **固定止损线**：入场价 - 2N（2倍ATR）
- **移动止损**：跟踪最高价，从最高点回撤2N自动触发
- 根据市场波动自动调整止损距离
- 波动大时止损距离远，波动小时止损距离近

#### 固定比例止损
- 亏损达到10%触发止损
- 作为海龟止损的补充保护
- 防止极端情况下的大幅亏损

### 3. 关键参数配置

```python
# 海龟交易法止损参数
g.turtle_atr_period = 20      # ATR计算周期
g.turtle_stop_n = 2           # 止损倍数(2N止损)
g.use_turtle_stop = True      # 是否启用海龟止损
g.use_fixed_stop = True       # 是否使用固定止损
g.fixed_stop_loss = 0.1       # 固定止损比例(10%)
```

### 4. 执行时间

- **14:00** - 第一次止损检查
- **14:30** - 第二次止损检查（确保及时止损）
- **15:10** - 打印持仓信息（含止损价位）

### 5. 通知机制

系统会通过飞书发送以下通知：
- ⚠️ 止损触发通知（含触发原因：海龟止损/固定止损）
- 显示当前收益率和最高回撤比例
- 每个持仓的止损价位实时更新

### 6. 止损逻辑说明

```
建仓时：
- 记录入场价格和入场时的ATR
- 计算初始止损价 = 入场价 - 2×ATR

持仓期间：
- 持续跟踪股价最高点
- 动态止损价 = max(初始止损价, 最高价 - 2×当前ATR)
- 同时检查是否触发10%固定止损

止损触发：
- 当前价 ≤ 动态止损价 → 海龟止损
- 亏损 ≥ 10% → 固定止损
```

### 7. 优势

1. **自适应性**：根据市场波动自动调整止损距离
2. **双重保护**：海龟止损+固定止损双保险
3. **保护利润**：移动止损可锁定部分利润
4. **风险可控**：最大亏损不超过10%

### 8. 参数调整建议

- **保守型**：`turtle_stop_n = 1.5`（更严格的止损）
- **标准型**：`turtle_stop_n = 2`（默认设置）
- **激进型**：`turtle_stop_n = 2.5`（给予更多波动空间）

可根据市场情况调整：
- 震荡市：减小turtle_stop_n至1.5
- 趋势市：可增加至2.5或3

## 注意事项

- 该策略需要在聚宽平台环境运行
- ATR计算需要至少21天的历史数据
- 止损检查每天14:00和14:30执行两次
- 建议先在模拟环境测试参数设置
